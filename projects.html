HashCrash

HashCrash is a password cracking tool I produced as I found many existing services yielded unreliable results when recently installed on new machines, unless the exact configuration was preserved often errors occurred. This program circumvents this by using a simple approach; the only file it requires is a list of possible passwords. Using this list it performs a dictionary attack against the hash provided by the user. This Attack works by reading the dictionary list line-by-line and performing a hash on each entry until the hash matches the one provided by the user. The program can determine which hashing function is being used by the length of the has provided. HashCrash on Github
